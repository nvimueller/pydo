# pydo

a tiny, zero-dependency todo list for your terminal.

do the task. mark it done. move on.

## install
`pip install pydo`

or clone and run directly:

`git clone https://github.com/yourname/pydo`
`cd pydo`
`python -m pydo`

## usage

`pydo add "write README"`
`pydo list`
`pydo done 1`
`pydo rename 2 "update README with examples"`

## commands

`pydo add "<task>"`              add a new task  
`pydo list`                      show all tasks with indexes  
`pydo done <index>`              mark a task as complete  
`pydo rename <index> "<task>"`   edit a task  

## why pydo

no config files, no cloud sync, no dependencies  
just 4 commands and a csv file
built for people who live in the terminal
